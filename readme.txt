## Description

This object adds **3 speed cameras** to the FiveM map, designed to blend realistically and immersively into the environment:

- **1 visible radar** — positioned on the side of the road, clearly identifiable
- **2 hidden radars** — camouflaged within scenery objects for added realism

The goal is to enhance the roleplay experience by providing credible speed checks without breaking immersion.
